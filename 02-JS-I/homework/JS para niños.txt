En JavaScript una variable es un nombre que contiene un valor o expresión. Cada vez que use ese nombre en su código, la computadora realmente usará el valor representado por ese nombre. La razón por la que se llaman"variables" es porque su valor puede cambiar durante la ejecución de un programa.
A esa variable que uno crea se le puede dar un valor con el operador de asignación: =.
Uno de los datos que podemos usar son las strings, ellas son cadenas de palabras encerradas entre comillas ".
Con las funciones podemos dar intrucciones para que hacer con las variables, se invocan poniendo la palabra Function y luego le damos un nombre a esa función, estamos obligados a abrir y cerrar un paréntesis redondo ( ) después del nombre personalizado. Dentro de ellos se ponen los argumentos, que pueden ser variables también, se pueden agregar tantos argumentos como se deseen, siempre separados de comas. La segunda cosa importante a tener en cuenta son los corchetes o llaves{}debajo de la primera línea. Entre estos corchetes tendremos la oportunidad de escribir una serie de instrucciones que se utilizarán automáticamente cuando se utilice nuestra instrucción principal. Podemos usar la declaración return para que devuelva un resultado de esa función.
El comando if, es una forma para ver si de darle instrucciones a una función si pasa algo correcto o verdadero o si pasa un resultado erroneo o falso.
Estos resultados verdaderos o falsos se llaman valores booleanos, significa que puede activar o desactivar una función por ejemplo, siendo true (verdadero) o false (falso)
